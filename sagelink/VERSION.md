# SageLink v0.2.1

SageLink — an E2E encrypted protocol suite built on SageLang.

## About

SageLink implements the **Noise_IK_25519_ChaChaPoly_BLAKE2s** handshake
pattern, providing mutual authentication, forward secrecy, and encrypted
transport over multiplexed streams.
